### ⚠️ ACTION REQUISE DE VOTRE PART

Vous devez uniquement créer la table dans la base de données :

1. Ouvrir **Supabase** → **SQL Editor**
2. Copier le contenu de `supabase/migrations/20260126_create_contrats_table.sql`
3. Exécuter le script
4. ✅ C'est tout !

---

## 🧪 COMMENT TESTER (2 min)

Une fois la table créée :

1. Aller dans l'application (elle va se recharger automatiquement)
2. Aller dans le menu **Chantiers** -> Sélectionner un chantier -> Onglet **Équipe**
3. Cliquer sur **"Ajouter Monteur"**
4. Ajouter un monteur permanent
5. Aller dans le nouveau menu **"Contrats"**
6. Vérifier que le contrat a été créé automatiquement 🎉
7. Cliquer sur l'icône de téléchargement pour obtenir le PDF du contrat 📄
